# Aurora Streaming

A streaming platform built with Next.js, inspired by Netflix. Browse movies and TV shows powered by the TMDB API, with Firebase authentication and personalized user profiles.

## Features

- **Browse & Discover** - Explore movies, TV shows, new & popular titles, and genre-based categories
- **Mood-Based Browsing** - Get recommendations based on your current mood
- **Search** - Find content across the entire TMDB catalog
- **Video Player** - HLS-powered player with episode navigation, sleep timer, and continue-watching support
- **User Profiles** - Multiple profiles per account with customizable avatars
- **My List** - Save titles to a personal watchlist stored in Firestore
- **Viewing Stats** - Track your watch history and habits with visual charts
- **Release Calendar** - See upcoming releases at a glance
- **Themes** - Switch between multiple UI themes
- **PWA** - Installable as a progressive web app for offline-ready access

## Tech Stack

- **Framework** - Next.js 16 with React 19
- **Styling** - Tailwind CSS 4 with Framer Motion animations
- **Auth & Database** - Firebase Authentication and Firestore
- **State Management** - Zustand
- **Data Fetching** - TanStack React Query with Axios
- **Video** - HLS.js
- **Charts** - Recharts
